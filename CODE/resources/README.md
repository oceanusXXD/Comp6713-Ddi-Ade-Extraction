Lightweight files used by the baseline scripts.

- `baseline/drug_lexicon.json`: drug lexicon
- `baseline/effect_lexicon.json`: effect lexicon
- `drugbank/drugbank_names.txt`: drug name list

## Used by

- `src.baseline.run_baseline`
