# Results

This folder stores summary-level evaluation outputs retained in the submission package.

## Included Files

- `quantitative_report.md`
- `quantitative_report.txt`
- `benchmark_suite_latest_raw_clean_base_20260329/summary.csv`
- `benchmark_suite_latest_raw_clean_balanced_e3_ckpt993_20260329/summary.csv`

## Interpretation

- The two `summary.csv` files are compact benchmark tables for the base model and the retained LoRA model.
- The quantitative report compares the base model and the retained LoRA adapter across internal and external datasets.

## Omitted Outputs

Per-sample benchmark predictions and full historical result folders are intentionally omitted from this package to keep the submission size manageable.
