# 报告目录说明

这个目录保存数据审计和训练过程的摘要文件。

## 文件说明

- `data_audit_report.md`
  数据审计报告，通常记录数据检查结果、污染排查和处理说明。
- `data_fix_log.md`
  数据修复日志，记录修复过哪些问题以及处理过程。
- `data_stats.json`
  数据统计结果的结构化 JSON 文件。
- `train_command.txt`
  曾经执行过的训练命令记录，便于复现。
- `train_summary.md`
  训练结果摘要，通常包含训练配置、关键观察和结果概览。

## 你什么时候会看这里

- 想确认数据到底改过什么
- 想知道当前主线是怎么从原始数据整理出来的
- 想快速回看上一轮训练命令和总结
